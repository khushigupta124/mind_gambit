Our Vision

The project MindgAmbIt is a web-based application that aims to revolutionize the classic game of Rock, Paper, Scissors by introducing an AI-driven component that can detect hand postures and make strategic predictions. This innovative game platform combines computer vision, machine learning, and interactive gameplay to provide users with a unique and engaging gaming experience.

Scope

Hand Posture Detection:

The core feature of MindgAmbIt is its ability to accurately detect and recognize hand postures in real-time using computer vision techniques. This allows players to physically mimic the gestures for Rock, Paper, or Scissors, adding a new dimension to the game.

AI Predictions:

The system incorporates a machine learning model that analyzes the detected hand postures and predicts the next move of the player. This AI-driven prediction adds an exciting challenge as players compete against a computer opponent with dynamic strategies.

Interactive Gameplay:

Users can play the game by simply making hand gestures, and the AI responds with its move based on predictions. The game interface is user-friendly and visually engaging, enhancing the overall gaming experience

Educational Aspect:

MindgAmbIt offers an educational component by providing insights into computer vision and machine learning concepts. Users can learn about these technologies while having fun playing the game.
